# Intelligent ID Card OCR System

## Overview

A production-style OCR system that extracts structured information from ID cards using computer vision and natural language processing techniques.

---

## Key Features

* OCR-based text extraction (EasyOCR)
* Intelligent text cleaning & normalization
* Rule-based + context-aware field extraction
* Bounding box visualization
* Confidence scoring
* Interactive UI (Streamlit)

---

##  System Pipeline

1. Image Upload
2. Text Detection (EasyOCR)
3. Text Cleaning 
4. Field Extraction 
5. Structured Output

---

##  Extracted Fields

* Name
* License Number
* Date of Birth
* Expiry Date
* Address

---
